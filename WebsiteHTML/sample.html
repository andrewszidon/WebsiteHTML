Sample Work: Android game project attempt.
During my Senior year, I took the opportunity of a directed studies class, where we design our own curriculum and assignments, to start a project for the sake of developing my programming skills, as I felt that my AP Computer Science course barely covered the core concepts of object oriented programming. This project has taught me many things. I have learned that code never runs the first or so time. I have learned to work in advance, so that there's enough time to get a response from StackOverflow. I have learned that I need to set the R/id of every UI interface object as I create them, because otherwise I'll have a bunch with the same name. I have learned that GitHub is helpful, and will make sure to use it for future projects. I have learned that saving and loading requires not only java code, but also something related to permissions, which is why I never got it to save. But I also learned many things from succeeding. I learned about how to create and interact with UI elements, both through xml and through code. I learned about how Android uses Activities and Layouts. And spinners.
